# AdorableArmory-1.20.1

> 一个正在缓慢制作中的 Minecraft Forge 1.20.1 模组。  
> 目前主要在制作 Boss、武器、方块、特效，以及一些自定义渲染系统。

### 版本与环境
- **Minecraft 版本**：`1.20.1`
- **模组加载器**：`Forge`
- **光影兼容测试**：`Oculus` + `Embeddium`

### 项目大概进度
- **Boss 内容**：`斯卡蕾特·萝拉·艾莉米娅` 正在缓慢制作中
- **武器内容**：`真魔之弓`、`真魔之箭` 已加入，后续还会继续完善表现
- **方块内容**：`LolaBlock` 已加入，并带有 Cosmic 星空渲染层
- **渲染系统**：Cosmic、SKY_ITEM、Item 描边、自定义附魔渲染已实现
- **光影兼容**：已针对 `Oculus` + `Embeddium` 做过兼容处理

### 目前的内容
- `AdorableArmory`（模组核心）
- `LolaBlock`（萝拉方块 / 暂命名，未来可变）
- `薄暮恋刃`（目前只是测试武器，未来可能会继续调整）
- `斯卡蕾特·萝拉·艾莉米娅` 刷怪蛋（Boss 刷怪蛋）
- `真魔之弓`（斯卡蕾特·萝拉·艾莉米娅的主要武器）
- `真魔之箭`（真魔之弓的弹药）
- `OrePerspectiveCore`（测试物品，目前功能是透视看见地下的任何矿石，未来可完善...）
- 更多内容还在慢慢制作中...

### 目前的实体
- **Boss**：`斯卡蕾特·萝拉·艾莉米娅`（RococoShin 最喜欢的一位 Boss，正在非常努力地设计她，目前正在缓慢制作...）
- **技术性实体**：主要用于 Boss 的特效、攻击、武器表现和一些特殊渲染

### 目前兼容部分（兼容光影模组 Oculus、Embeddium）
- 描边、自定义附魔渲染兼容光影（2026/04/24）
