# AdorableArmory (1.20.1)

> 一个正在缓慢制作中的 Minecraft Forge 1.20.1 模组。  
> 目前主要致力于开发 **Boss**、**武器**、**方块**、**特效**，以及**自定义渲染系统**。

## 🛠️ 版本与环境

| 属性 | 详情 |
| :--- | :--- |
| **Minecraft 版本** | `1.20.1` |
| **模组加载器** | `Forge` |
| **模组前置** | 目前没有 |
| **光影兼容** | `Oculus` + `Embeddium` |

## 🚀 开发进度与特性

- **Boss 开发**：`斯卡蕾特·萝拉·艾莉米娅` 正在缓慢制作中...
- **武器**：`真魔之弓`、`真魔之箭` 已加入，后续会继续完善...
- **方块**：`LolaBlock` 已加入，并带有 `Cosmic` 星空渲染层。
- **渲染**：支持 Cosmic、SKY_ITEM、Item 描边渲染、自定义附魔渲染等。
- **光影兼容**：已针对 `Oculus` + `Embeddium` 做过专门的兼容处理。
- ⚡ 次元斩（跨模组联动）：正在制作与完善中（与 Astralrail Cube 模组联动，并协助其作者 Seraphina（带长）定制开发该特效）。

## 📦 当前内容一览

### 🔧 核心与方块
- `AdorableArmory`：模组核心
- `LolaBlock`：萝拉方块（暂命名，未来可变）

### 🗡️ 武器与物品

- `薄暮恋刃`：目前为测试武器（未来可变）

### 👾 实体图鉴

- **Boss**：`斯卡蕾特·萝拉·艾莉米娅`（RococoShin 最喜欢的一位 Boss，正在非常努力地设计中，目前制作中...）
