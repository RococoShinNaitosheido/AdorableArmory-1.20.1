# AdorableArmory-1.20.1

> 一个正在缓慢制作中的 Minecraft Forge 1.20.1 模组。  
> 目前主要在制作 Boss、武器、方块、特效，以及一些自定义渲染系统。

### 版本与环境
- **Minecraft 版本**：`1.20.1`
- **模组加载器**：`Forge`
- **光影兼容**：`Oculus` + `Embeddium`

### 项目大概进度
- **Boss**：`斯卡蕾特·萝拉·艾莉米娅` 正在缓慢制作中...
- **武器**：`真魔之弓`、`真魔之箭` 已加入，后续会继续完善...
- **方块**：`LolaBlock` 已加入，并带有 `Cosmic` 星空渲染层
- **渲染**：Cosmic、SKY_ITEM、Item描边渲染、自定义附魔渲染等等
- **光影兼容**：已针对 `Oculus` + `Embeddium` 做过兼容处理

### 目前的内容
- `AdorableArmory`（模组核心）
- `LolaBlock`（萝拉方块 / 暂命名，未来可变）
- `薄暮恋刃`（目前只是测试武器，未来可变）
- `斯卡蕾特·萝拉·艾莉米娅` 刷怪蛋（Boss 刷怪蛋）
- `真魔之弓`（斯卡蕾特·萝拉·艾莉米娅的主要武器）
- `真魔之箭`（真魔之弓的箭）
- `OrePerspectiveCore`（测试物品，目前功能是透视看见地下的任何矿石，未来考虑完善...）
- 更多内容还在制作中...

### 目前的实体
- **Boss**：`斯卡蕾特·萝拉·艾莉米娅`（RococoShin 最喜欢的一位 Boss，正在非常努力地设计她，目前正在制作...）
- **技术性实体**：主要用于 Boss 的特效、攻击等等...

### 兼容部分（兼容光影模组 Oculus、Embeddium）
- 描边、自定义附魔渲染兼容光影（2026/04/24）
- Cosmic星空渲染兼容 Oculus shaderpack（2026/05/1）

### 未来变动
- `LolaBlock`后面会改名，因为它后面会有不同的用途
- `薄暮恋刃`后面会作为Boss的主要武器
